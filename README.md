"곡괭이로 파낸 디미고 최고 자리" 게임 규칙 설명서

1. 게임 목표

이 게임의 최종 목표는 엄청난 돈(Cr)을 벌어 '디미고 교장'과 '디미고 이사장' 업그레이드를 구매하는 것입니다.

이를 위해 당신은 황무지에 자동화 공장을 건설하고, 자원을 캐서 가공하고, 완성된 물품을 팔아 수익을 내야 합니다.

2. 기본 조작

W, A, S, D: 화면(카메라) 이동

마우스 좌클릭: 건물 설치 / UI 버튼 클릭

마우스 우클릭: 건물 정보 확인 / 건물 UI 열기 (보관함, 용해로, 연구소 등)

1~9 또는 마우스 휠: 핫바(하단 단축 슬롯) 선택

R: 설치할 건물 회전

F: 주변 바닥에 떨어진 아이템 줍기

X: 삭제 모드 켜기/끄기 (삭제 모드에서 좌클릭으로 건물 철거)

E: 인벤토리 열기

B: 상점 열기

M: 미션 목록 열기

T: 연구 트리 열기 (연구소 우클릭 시)

H: 게임 설명서 열기

3. 게임의 기본 흐름 (자동화 공장)

게임은 "채굴 → 운송 → 제련 → 조립 → 판매/연구"의 흐름으로 진행됩니다.

채굴: 맵에 있는 자원(철, 구리, 석탄, 나무) 위에 채굴기를 설치하면 자동으로 자원을 캐냅니다.

운송: 컨베이어를 설치해 채굴된 아이템을 원하는 곳으로 옮깁니다.

제련: 용해로를 설치하고, 석탄/나무(연료)와 광석(철/구리)을 투입하면 주괴가 생산됩니다.

조립: 제작기를 설치하고 레시피(예: 철 톱니바퀴)를 선택한 뒤, 재료(예: 철 주괴)를 투입하면 부품이 생산됩니다.

판매: 생산된 아이템이 판매기로 들어가면 자동으로 돈(Cr)으로 바뀝니다.

연구: 자동화 팩(제작기에서 생산)을 연구소에 넣으면 새로운 기술을 잠금 해제할 수 있습니다.

4. 핵심 건물 가이드

채굴기: 자원 위에 설치 시, 바라보는 방향(화살표)으로 아이템을 배출합니다.

채굴기: 1x1 범위 채굴

산업용 채굴기: 3x3 범위 채굴 (연구 필요)

용해로 (우클릭): 광석을 주괴로 만듭니다.

왼쪽 상단 (재료 슬롯): 철광석 또는 구리광석을 넣습니다.

왼쪽 하단 (연료 슬롯): 석탄 또는 나무를 넣습니다.

오른쪽 (결과물 슬롯): 완성된 주괴가 나옵니다.

아이템이 컨베이어로 투입될 때 재료/연료를 알아서 구분해 받습니다.

제작기 (우클릭): 부품을 조립합니다.

오른쪽 메뉴: 만들고 싶은 아이템(레시피)을 먼저 선택해야 합니다.

왼쪽 (재료 슬롯): 레시피에 필요한 재료를 넣습니다. (최대 4종류)

오른쪽 (결과물 슬롯): 완성된 부품이 나옵니다.

바라보는 방향으로 완성품을 배출합니다.

연구소 (우클릭 → T): 기술을 연구합니다.

자동화 팩을 컨베이어로 투입하거나 직접 넣습니다.

연구소를 우클릭해 연구 트리를 연 뒤, 원하는 기술을 선택하면 연구소가 자동으로 팩을 소모하며 연구를 진행합니다.

분류기 (우클릭): 아이템을 필터링합니다.

중앙 (필터 슬롯): 분류할 아이템을 등록합니다. (예: 철 주괴)

일치 시: 필터와 일치하는 아이템(철 주괴)이 나갈 방향(왼쪽/직진/오른쪽)을 설정합니다.

불일치 시: 필터와 다른 모든 아이템이 나갈 방향(왼쪽/직진/오른쪽)을 설정합니다.

아이콘: 건물 위에 설정된 방향이 아이콘으로 표시됩니다. (불일치 시: 필터 아이템 + X 표시)

5. 경제와 성장

상점 (B키)

건물/업그레이드: 각종 건물이나 업그레이드를 돈(Cr)으로 구매합니다.

자원 판매: 인벤토리에 있는 자원을 직접 판매할 수 있습니다. (모두 판매 버튼 권장)

패시브 수입: 디미고 입학권, 학생회장 등 "업그레이드" 아이템을 구매하면, 1초마다 자동으로 돈이 들어옵니다. (Cr/s)

미션 (M키)

제시된 미션의 요구 아이템을 모아 완료 버튼을 누르세요.

보상으로 돈이나 영토 확장을 얻을 수 있습니다.

영토 확장: 공장을 지을 수 있는 땅(초록색 잔디)이 넓어집니다.

연구 (연구소 우클릭 → T키)

자동화 팩을 소모하여 고속 컨베이어, 산업용 채굴기 등 더 좋은 건물을 잠금 해제할 수 있습니다.
